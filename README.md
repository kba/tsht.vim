tsht.vim
========
Vim Syntax highlighting for [tsht](https://github.com/kba/tsht) unit tests.

<!-- vim :GenTocGFM -->

* [Installation](#installation)
	* [Pathogen](#pathogen)
	* [Vundle](#vundle)

## Installation

### Pathogen

```sh
cd ~/.vim
git clone 'https://github.com/kba/tsht.vim' bundle/tsht.vim
```

### Vundle

Add to your `.vundlerc`:

```vim
Plugin 'kba/tsht.vim
```
